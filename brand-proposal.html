05 â€” Visual Language

Less is the system

Iconography

Line icons, 1.5px stroke

Rounded caps/joins for warmth. Lucide as base set, custom for Trawly-specific concepts.

Illustration

None (by design)

Brand strength from typography, colour, and information design. Exception: one abstract data-art hero for landing page.

Photography

Editorial only

If needed: ocean textures, data viz, workspace scenes. Warm filter always. Never cheesy stock.

Motion

Calm and confident

Fade up + translate (8px), 300ms ease. No parallax, no bounce, no spring physics.

Spacing

4px base grid

Scale: 4, 8, 12, 16, 24, 32, 48, 64. Generous whitespace â€” density of content, not pixels.

Layout
