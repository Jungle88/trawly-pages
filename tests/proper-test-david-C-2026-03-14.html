Google & Microsoft Shipped WebMCP in Chrome 146 — Every Website Is Now an AI Tool
WebMCP is a new web standard that lets AI agents interact with websites as structured tools — no scraping, no browser automation hacks. Google and Microsoft shipped it in Chrome 146 this week. Sites can now expose a JSON "Tool Contract" that agents query directly. W3C is formalizing it the same way they formalized HTML5. Today it's optional. In a few years, it won't be.
